# Color Verse SEO Setup

Files included:
1. robots.txt
2. sitemap.xml
3. seo-head-tags.html

How to use:

1. Upload robots.txt and sitemap.xml to your website root folder.

2. Copy the content from seo-head-tags.html into the <head> section of your main HTML file.

3. Replace YOUR_CODE with your Google Search Console verification code.

4. Redeploy your website on Vercel.

5. Submit sitemap:
https://color-verse-theta.vercel.app/sitemap.xml

6. Open Google Search Console:
https://search.google.com/search-console/about
